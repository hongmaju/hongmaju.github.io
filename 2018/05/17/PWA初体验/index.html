发表于 2018-05-17

一：什么是PWA

  • PWA（ 全称：Progressive Web App ）也就是说这是个渐进式的网页应用程序。它主要有三个技术：
    • Service Worker（ ps：就叫做中间服务商 ）
    • Manifest （应用清单，网页应用程序的图标、是否全屏，启动动画等配置文件）
    • Push Notification（网页应用程序的推送通知）
